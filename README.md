## Application de gestion de relevés – Projet SI & IA

Ce dépôt contiendra une application web de **gestion de relevés de notes** intégrant un module simple d’**intelligence artificielle** (analyse du risque d’échec, moyenne prévisionnelle, etc.).

L’objectif du projet est de couvrir les principales étapes du **cycle de vie d’un système d’information** : analyse, conception, implémentation, tests et déploiement simplifié.

---

## Stack technique prévue

- **Backend** : Node.js, Express, ORM (Sequelize ou Prisma), JWT, base de données SQL (PostgreSQL ou MySQL).
- **Frontend** : React.js, React Router, Axios, bibliothèque UI (Material UI ou Bootstrap).

> La structure du code (dossiers `server` et `client`) sera ajoutée au fur et à mesure de l’avancement.

---

## Prérequis

- Node.js (version LTS recommandée)
- NPM ou Yarn
- Un SGBD SQL (PostgreSQL / MySQL / MariaDB, selon le choix du groupe)

---

## Installation (esquisse)

Les étapes exactes seront complétées quand les projets `server` et `client` seront créés, mais l’idée générale sera :

```bash
# Cloner le projet
git clone <URL_DU_DEPOT>
cd <nom_du_dossier>

# Backend (à venir)
cd server
npm install
npm run dev

# Frontend (à venir, dans un autre terminal)
cd client
npm install
npm start
```
